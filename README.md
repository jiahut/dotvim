dotvim
======
+ use
    - cd ~
    - cp -r .vim vim.bak;cp .vimrc vim.bak
    - git clone git@github.com:jiahut/dotvim.git
    - ln -s dotvim/.vim .vim
    - ln -s dotvim/.vimrc .vimrc
+ install
    - cd ~/.vim/bundle
    - git clone https://github.com/tpope/vim-sensible.git
    - or unzip ~/download/snipMate.zip -d .
